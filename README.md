# go-nextjs-fullstack-ddd-skill

Production-ready Claude Code skill for building full-stack applications with:

- Go backend
- Next.js + shadcn/ui frontend
- shadcn MCP-assisted UI selection
- PostgreSQL
- Docker Compose startup with `docker-compose up -d`
- DDD project structure
- OpenAPI-first contract design
- Generated Go server contracts and Next.js TypeScript client
- Premium frontend quality with `frontend-design` then `ui-ux-pro-max`

## Included Skill

- `go-nextjs-fullstack-ddd/`
- Packaged zip: `go-nextjs-fullstack-ddd.zip`

## Install

Copy the `go-nextjs-fullstack-ddd/` directory into your Claude skills directory, or use the packaged zip from the GitHub release assets.
